# Leveragents Skills Marketplace

AI skills for ecommerce operators. Built for Claude Code.

## Install

```bash
# Add the marketplace
/plugin marketplace add benjraju/leveragents-skills

# Install any skill
/plugin install ad-creative-generator@leveragents
/plugin install amazon-listing-optimizer@leveragents
/plugin install ecom-review-analyzer@leveragents
/plugin install email-sequence-builder@leveragents
/plugin install social-content-generator@leveragents
```

## Available Skills

| Skill | Description | Price |
|-------|-------------|-------|
| **Ad Creative Generator** | Generate 40+ ad creatives from a product URL using Gemini API | Free |
| **Amazon Listing Optimizer** | Optimize titles, bullets, descriptions for A9 search ranking | Free |
| **Review Analyzer** | Extract insights and pain points from customer reviews | Free |
| **Email Sequence Builder** | Build abandoned cart, post-purchase, win-back email flows | Free |
| **Social Content Generator** | Platform-specific social posts, threads, and video scripts | Free |

## How It Works

Each skill is a Claude Code plugin containing instructions that Claude follows when you invoke them. Drop them in and Claude instantly knows how to execute ecommerce tasks.
